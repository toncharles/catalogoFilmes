Bem Vindo ao meu APP Ionic!


A documentação do framework Ionic apresenta vários componentes úteis que são incluídos fora da caixa com ionic.

O que é Ionic?


O Ionic Framework é um SDK de software livre que permite aos desenvolvedores criar aplicativos móveis de alta qualidade com tecnologias da Web, como HTML, CSS e JavaScript.

O que é Ionic Cloud?


O Ionic Cloud é uma plataforma em nuvem para gerenciar e dimensionar aplicativos Ionic com serviços integrados, como notificações push, compilações nativas, autenticação de usuário e atualização ao vivo.
